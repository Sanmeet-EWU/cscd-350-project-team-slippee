Guide

Welcome to the Project Slippe 64 guide! This guide will help you understand how to use the application effectively.

Getting Started

  • First, select which emulator you want to convert from, and which you want to convert to

  • Next, you'll need to pick your current save files. The number of files and their locations will vary depending on the emulator you're currently using

Ares

  • Ares saves are kept in the same directory as the ROM you made the save with
  • The files will have the same name as the ROM, with different extensions
  • There may be 1 or 2 files, upload whatever you find

Bizhawk/Emuhawk

  • Go to Emuhawk's file location (use the desktop or start menu shortcut) {% load static %}
  • Go to N64/SaveRAM within Emuhawk's folder {% load static %}
  • The file name will be ROM_name.saveRAM (e.g. Super_Mario_64.saveRAM)

Retroarch

  • Go to Retroarch's file location (use the desktop or start menu shortcut) {% load static %}
  • Go to the saves folder
    • {% load static %}
  • From here, go to the folder named after the core you saved your game in {% load static %}
  • The file name will be ROM_name.SRM

Translating

  • Once you've selected the right emulators and files, press the 'submit all' button
  • The application will process your files and convert them to the desired format
  • A .zip file will then be downloaded to your computer
  • simply uncompress these files and move them to the appropriate emulator folder
  • your save game should now be playable on the new emulator
